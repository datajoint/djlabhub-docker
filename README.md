# Base DataJoint JupyterLab Image for JupyterHub

## Launch locally


`docker-compose -f dist/alpine/docker-compose.yml --env-file config/.env up --build`
OR
`docker-compose -f dist/debian/docker-compose.yml --env-file config/.env up --build`
